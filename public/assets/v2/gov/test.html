Welcome!

Does this page look like being hosted on http://www2.gov.bc.ca? If you think so, then my goal is partially achieved! Actually, the page is generated from a static HTML file of which source is available here. Looking at the source, you'll find it only contains this "Welcome" paragragh. Then where does the header (all stuff before this paragragh) and footer (all stuff afterwards) come from? The answer is this JS block: 
<script type="text/javascript" src="/v2/gov/"></script>
<script type="text/javascript">
    unippear({
        headerContainer: '#wrappe',
        footerContainer: '#wrapper',
    });
</script>

Additonal options when calling unippear can be found here.

Factoring header, footer and associated assets (CSS, JS, img etc) out of a document isn't new. Almost all template engines are capable of. But those template engines are platform specific and thus only work for one website. Unipear turns the layout into a service to make it shareable among multiple websites. If this is of interest to you, return to MyGovBC-core-headerFooterService and keep reading. You may also find Unippear docs to be useful.
